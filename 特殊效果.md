#### 文本缩略显示

- 单行文本缩略显示

```css
.a {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}
```

- 多行文本缩略显示

```css
/* 方法一：不兼容safari */
.a {
  display: -webkit-box;
  /*。指定显示几行后缩略 */
  -webkit-line-clamp: 3;
  -webkit-box-orient: vertical;
}

/* 方法二：需要限制高度 */
.a {
  overflow: hidden;
  text-overflow: ellipsis;
  max-height: 5em;
}
```
